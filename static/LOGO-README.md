# SpentWorth Logo Files

## Social Media Profile Images

Use these SVG files for your social media profiles:

- `logo-512.svg` - 512x512 version (good for most social media platforms)
- `logo-1024.svg` - 1024x1024 version (high resolution for larger displays)

### Converting to PNG

To convert these SVG files to PNG format:

1. **Using an online converter:**
   - Go to https://cloudconvert.com/svg-to-png
   - Upload the SVG file
   - Download the PNG

2. **Using a browser:**
   - Open the SVG file in Chrome/Firefox
   - Right-click and "Save As" or take a screenshot
   - Or use browser dev tools to export

3. **Using command line (if you have ImageMagick):**
   ```bash
   convert logo-512.svg logo-512.png
   convert logo-1024.svg logo-1024.png
   ```

### Recommended Sizes by Platform

- **Twitter/X:** 400x400 minimum (use 512x512)
- **Instagram:** 320x320 minimum (use 512x512)
- **LinkedIn:** 300x300 minimum (use 512x512)
- **Facebook:** 180x180 minimum (use 512x512)
- **Reddit:** 256x256 (use 512x512)

All platforms support PNG format. The square 1:1 aspect ratio works for all major social media platforms.
